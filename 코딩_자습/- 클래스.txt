- 클래스
새로운 형식을 정의 ➡ 객체를 만드는 틀
클래스 이름은 CamelCase로 작성

- 객체
속성을 가질 수 있는 모든 것 ➡ 클래스로부터 만들어진 것, 인스턴스라고도 함
속성: 클래스 또는 객체에 소속된 변수 or 함수
	필드(field): 클래스 혹은 객체에 소속된 변수
	메소드(method): 클래스 혹은 객체에 소속된 메소드

클래스는 붕어빵 틀이고, 객체가 붕어빵이라고 볼 수 있음
하나의 클래스로 여러 개의 객체를 만듦
같은 클래스로부터 생성된 객체들끼리는 서로 영향을 주지 않음

클래스 메소드와 함수의 차이

메소드 정의시 self 변수가 항상 매개변수 목록의 맨 앞에 위치
정의: 메소드(self, 매개변수1, 매개변수2, …)
O 매개변수가 없을 때도 self는 꼭 작성
O 현재 객체 자신을 의미 ➡ self라는 이름을 가장 많이 씀

호출: 메소드(매개변수1, 매개변수2)

EX)
1. Cookie 클래스 생성 ➡ firstcookie 객체 생성

2. 메소드 호출 ➡ firstcookie.method(매개변수1, 매개변수2)

3. 파이썬 ➡ Cookie.method(firstcookie, 매개변수1, 매개변수2)
